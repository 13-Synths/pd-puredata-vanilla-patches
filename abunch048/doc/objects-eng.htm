AUDIO EFFECTS

phaser : a phaser effect
simple-delay : a simple delay effect with a maximum delay of 1 second
long-delay : a delay effect with a maximum delay of 10 seconds
feed-delay : a delay with an 'infinite' feedback
reverb : a reverb effect
simple-chorus : a chorus effect
simple-pitchshift : transpose the pitch of a sound stream
envelopes : 8 different envelopes to cut a short sound from a sound stream
simple-filter : a bandpass filter with 8 bands
8band-filter : an expanded version of the previous 'simple-filter' object
lowhigh-filter : a lowpass and an highpass filter
disto-filter : a distortion effect based on a filter
old-vocoder : a 'traditional' vocoder effect
matrix4 : a matrix or switch to distribute 4 sound inputs to 4 outputs
panning : distribute one audio channel (mono) to 2 channels (stereo)
al-disto : a gain and distortion effect

SOUND SOURCE

waves : 6 different wave forms (oscillators) and a noise generator
waves-add : adjust 16 harmonics and synthesize them in one wave form (additive synthesis)
play-file : play a sound file from the hard disk
record-file : record a sound to a new file on your disk
play-sample : play a short sample from the computers memory
simple-sampler : play a short sample from the computers memory by sending pitch values to its inlet
record-sample : record a short sound fragment that can be used immediately in other abunch objects like play-sample or grain-sample
scratch-sample : scratch a sample like a vinyl record
crackle : some basic digital 'cracking' sounds
sound-out: send the sound to the sound output of the computer
addy : an additive synthesizer
ozzy : a classical synthesizer with oscillators, filters and envelopes
tarra : a synthesizer based on a string model
crynth : a rather strange wavetable synthesizer
grain-sample : granular synthesis of a sound sample (pitch and length of the grains are independent)
grain48-sample : the same as grain-sample but with maximum 48 simultaneous grains
graino-sample : granular synthesis of a sound sample (length and traject influence the transposition of the grains)
grain-synth : a granular synthesizer
grain-live : a first (unfinished) version of a granular delay
wa-synth : a simple waveshaping synthesizer
simple-fm : a very basic object to make FM sounds
2mod-fm : FM synthesis with 2 modulators
syna-fm : an FM synthesizer (with 2 operators)
cross-fm : a second FM synthesizer
wind-synth : an simple synthesizer based on filtered noise

CONTROL

timeline : draw values in a table (f.e. to control a slider in an other object)
sequencer : a simple sequencer with maximum 16 steps (with on/off toggles for every step)
sequencer : a simple sequencer with maximum 16 steps (with number boxes for every step)
multi-seq : a sequencer with 6 tracks and 16 patterns (with on/off toggles for every step)
multi-seqn : a sequencer with 6 tracks and 16 patterns (with number boxes for every step)
keyboard-azerty : use your azerty computer keyboard as a simple piano (midi) keyboard
keyboard-qwerty : use your qwerty computer keyboard as a simple piano (midi) keyboard
clock : a clock with adjustable tempo (to synchronize abunch objects like timeline or sequencer)
multi-clock : a clock with 6 clock signals (one main clock and 5 other ones in an adjustable ratio to this main clock)(to synchronize abunch objects like timeline or sequencer)
random-out : send out random numbers
random-walk : send out numbers from a randomly a- or descending curve
scales : adjust a pitch to a chosen scale
scales-div : divide an octave in any number of tones
play-score : play a score (in txt format)
record-score : record a stream of controle data to a txt file
midi-ctlin : receive values from your midi controller hardware
metronome : show a tempo and/or hear a metronome tick
o-scope : an oscilloscope to view the wave form of an audio stream
e-scope : show the curve of an event stream (not audio)
spectrum : watch the spectrum of a sound
analyse : analyse a sound stream (harmonics, pitch, rhythm)

PRESETS

presets : store the position of faders, the values of number boxes, the name of sound files, etc. (in abunch objects) as a preset file

OBSOLETE

text-score : changed the name of this object to play-score

slice : replaced by the abunch object envelopes
matrix4-in and matrix4-out : replaced by the abunch object matrix4
